wyReferee

Welcome to wyReferee client. This client allows you to create, manage, publish and import both mappools and tournaments. It has a lot of QoL features for osu! multiplayer lobbies, such as picking maps from a mappool with just a single click on a button.

How to use the client

Visit Github for a detailed guide on how to use this client. Know your way around already, but forgot how to set it all up again? Simply follow the steps listed below and you're good to go!

Help! I found a bug/I want to request a new feature

Head on over to Github and create an issue.

Getting started

Head on over to Settings where all the general settings are located.

osu! authentication

This is used to identify who you are. A new window will pop up where you will have to give permission for wyReferee to read public data on your behalf. This basically means that wyReferee will be able to tell who is logged in, which then will be used to give you permissions to create, edit and view tournaments.

osu! irc

Probably the single most important feature of this client. The irc connection is used to create multiplayer lobbies, as well as communicate in those multiplayer lobbies. Get your irc credentials by clicking on the link just below the Connect button.

osu!api key

This client requires you to set your api key to get various data from the osu server. In order for the client to show everything properly, we have to get data from the osu! servers. This way we will be able to get all multiplayer match data and beatmap data.
