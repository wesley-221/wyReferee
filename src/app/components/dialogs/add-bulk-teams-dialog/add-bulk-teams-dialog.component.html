Add bulk teams

Enter one team per line using the following format:

Team name, Player 1 name, Player 1 id, Player 2 name, Player 2 id, ...
  • Team name is required
  • You can add multiple players per team, or none at all
  • Each player must follow the format: username, user ID
  • If a player is added, both the username and user ID are required
    • However, the user ID may be left empty if unknown. See the last example.
  • Separate each player with a comma

Examples:

Team 1, Player 1, 1234567, Player 2, 7654321
Team 2, Player 4, 1234567, Player 5, 7654321, Player 6, 1726354
Team 3, Player 7, 1234567
